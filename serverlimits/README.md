# ServerLimits

Show the current Discord server caps for the guild where the command runs.

## Commands

- `[p]serverlimits` - Open a dashboard with live usage bars and tier-based caps.

## Notes

- Metrics are based on Discord's "Discord Account Caps, Server Caps, and More" support article.
- Some values are derived from the current guild state and boost tier.
- Invite totals only show when the bot can view invites.
